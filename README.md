# Kuresel-Altin-Fiyatlari-Zaman-Serisi-Analizi-1990-2021-
Bu depo, 1990–2021 döneminde farklı ülkelerin aylık altın fiyat verilerini kullanarak veri hazırlama, Power BI görselleştirmeleri, volatilite ve korelasyon incelemeleri, kümeleme ve zaman serisi tahminleri (ARIMA, SARIMA, Holt‑Winters, Prophet vb.) içeren kapsamlı bir analitik projenin tüm dosyalarını barındırır.
